**⚡ JobHuntly** — An AI-powered job intelligence platform that scrapes Glassdoor & LinkedIn with stealth automation, ranks listings against your resume using NLP, and fires personalised daily alerts through a fully async pipeline.

| Feature | Detail |
|---|---|
| 🕷️ **Stealth Scraping** | `undetected-chromedriver` — bypasses Glassdoor & LinkedIn bot detection |
| 🧠 **Resume Matching** | TF-IDF + cosine similarity via PyMuPDF — ranks jobs against your PDF resume |
| ⚡ **Live Updates** | Real-time WebSocket progress via Django Channels |
| 🕐 **Auto Pipelines** | Celery Beat — daily scrape + personalised email at midnight |
| 🔐 **Auth System** | OTP activation, JWT sessions, password reset, custom user model |
| 🐳 **Dockerised** | Multi-service Compose: Postgres · Redis · Celery worker + beat |
